# deadlock-mcp

MCP server em TypeScript para consultar partidas, performance e itens via `api.deadlock-api.com`.

## Setup

```bash
pnpm install
```

O projeto usa o repositĂłrio `openapi-clients` como submodule em `openapi-clients/`.
Os scripts `dev` e `build` ja executam `setup:clients` automaticamente.

## Rodar

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```
